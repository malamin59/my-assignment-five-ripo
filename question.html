Question-2: What is the difference between innerHTML, innerText, and textContent ?

Key Differences:
1. innerHTML → Gets or sets HTML content (includes tags).
2. innerText → Gets or sets only visible text (ignores hidden elements).
3. textContent → Gets or sets all text (including hidden text but no HTML).

Question-3: What is event delegation in the DOM?

Definition:
Event delegation is a technique where a single parent element handles events for multiple child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Definition:
Event bubbling is a propagation method in the DOM where an event starts from the target element and moves up to its parent elements.
